What’s new in PyMC3 3.1¶

We are about to release a new version of PyMC3 after we released the first stable version 3.0 in January. Since then, we have seen a huge increase in contributions and thus added many exciting new features of which I want to highlight a few.

Z = np.linspace(0,3,100)[:,None]

with pm.Model() as model:
    # priors on the covariance function hyperparameters and noise
    l = pm.Uniform('l', 0, 10)
    log_s2_f = pm.Uniform('log_s2_f', lower=-10, upper=5)
    log_s2_n = pm.Uniform('log_s2_n', lower=-10, upper=5)
    f_cov = tt.exp(log_s2_f) * pm.gp.cov.ExpQuad(1, l)

    # Instantiate GP
    y_obs = pm.gp.GP('y_obs', cov_func=f_cov, sigma=tt.exp(log_s2_n),
                     observed={'X': X, 'Y': y})

    trace = pm.sample(2000, tune=1000)

    # Draw samples from GP
    gp_samples = pm.gp.sample_gp(trace[1000:], y_obs, Z, samples=50, random_seed=42)

Improvements to NUTS¶

Several people have also contributed many improvements to NUTS. It is now identical to Stan’s implementation and also much much faster. In addition, Adrian Seyboldt added higher-order integrators, which promise to be more efficient in higher dimensions, and sampler statistics that help identify problems with NUTS sampling.

Variational Inference¶

Maxim “Ferrine” has done outstanding contributions to improve support for Variational Inference. I would still label most of this as experimental, but it’s very promising. Essentially, Ferrine has implemented Operator Variational Inference (OPVI) which is a framework to express many existing VI approaches in. He has also made it much easier to supply mini-batches. See here for a full example of a Bayesian NN.
